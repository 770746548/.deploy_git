第6章 动画

6-1 Vue中的css动画原理

我们给transition name属性定义的是fade 所以是下面名称：
fade-enter fade-enter-to fade-enter-active
fade-leave fade-leave-to fade-leave-active
如果我们没有给transition定义name属性，用默认的那么就是：
v-enter v-enter-to v-enter-active
v-leave v-leave-to v-leave-active

进场动画原理

刚开始存在fade-enter和fade-enter-active
紧接第二帧的时候，fade-enter消失、fade-enter-to 出现
到最后的时候fade-enter-to消失、fade-enter-active消失

离开动画原理

刚开始存在fade-leave和fade-leave-active
紧接第二帧的时候，fade-leave消失、fade-leave-to 出现
到最后的时候fade-leave-to消失、fade-leave-active消失

  1. //css动画效果（css过度动画效果）
  2. <style>
  3.     .fade-enter {
  4.       opacity: 0;
  5.     }
  6.     .fade-enter-active {
  7.       transition: opacity 1s;
  8.     }
  9.  
  10.     .fade-leave-to {
  11.       opacity: 0;
  12.     }
  13.     .fade-leave-active{
  14.       transition: opacity 1s;
  15.     }
  16.   </style>
  17.  
  18. <div id="root">
  19.     <transition name="fade">
  20.       <div v-if="show">hello world</div>
  21.     </transition>
  22.  
  23.     <button @click="handleClick">toggle</button>
  24.   </div>
  25.   <script>
  26.     var app = new Vue({
  27.       el: '#root',
  28.       data: {
  29.         show: true
  30.       },
  31.       methods: {
  32.         handleClick: function () {
  33.           this.show = !this.show;
  34.         }
  35.       }
  36.     })
  37.   </script>

ps:
显示操作
刚开始fade-enter opacity为0 第二帧fade-enter消失 opacity变为1 这个过程一直在fade-enter-active 监听1秒时间
隐藏操作
刚开始fade-leave opacity 默认是1 第二帧 fade-leave消失 fade-leave-to出现 opacity 变为0 这个过程一直在fade-leave-active监听1秒时间后消失

第7章 路由

7-1.什么是路由

路由：就是我们通过不同的 URL 访问不同的内容。

7-2.Vue 路由的安装

  • cdn
    https://unpkg.com/vue-router/dist/vue-router.js
  • npm

npm install vue-router

7-3.Vue 路由的简单案例

  1. <!DOCTYPE html>
  2. <html lang="en">
  3.  
  4. <head>
  5.   <meta charset="UTF-8">
  6.   <meta name="viewport" content="width=device-width, initial-scale=1.0">
  7.   <meta http-equiv="X-UA-Compatible" content="ie=edge">
  8.   <script src="https://cdn.jsdelivr.net/npm/vue"></script>
  9.   <script src="https://cdn.bootcss.com/vue-router/2.8.1/vue-router.min.js"></script>
  10.   <title>路由</title>
  11.   <style>
  12.     ._active {
  13.       color: red
  14.     }
  15.  
  16.     ._activeTwo {
  17.       color: pink
  18.     }
  19.   </style>
  20. </head>
  21.  
  22. <body>
  23.   <div id="root">
  24.     <h1>hello 路由!</h1>
  25.  
  26.     <!-- 使用 router-link 组件来导航. -->
  27.     <!-- 通过传入 `to` 属性指定链接. -->
  28.     <!-- <router-link> 默认会被渲染成一个 `<a>` 标签 -->
  29.     <p>
  30.       <router-link to="/header">go header</router-link>
  31.       <router-link to="/footer">go footer</router-link>
  32.     </p>
  33.  
  34.     <!-- <router-link> replace 和 append的应用 标签 -->
  35.     <p>
  36.       <router-link :to="{path:'/header'}" replace>go header(replace)</router-link>
  37.       <router-link :to="{path:'footer'}" append>go footer(append)</router-link>
  38.     </p>
  39.  
  40.     <!-- <router-link> tag 渲染 -->
  41.     <p>
  42.       <router-link :to="{path:'/header'}" tag="li">go header(tag)</router-link>
  43.     </p>
  44.  
  45.     <!-- <router-link> exact-active-class 和 active-class 渲染 -->
  46.     <p>
  47.       <router-link :to="{path:'/header'}" exact-active-class="_active">go header(exact-active-class)</router-link>
  48.       <router-link :to="{path:'/footer'}" active-class="_activeTwo">go header(active-class)</router-link>
  49.     </p>
  50.  
  51.     <!-- <router-link> event渲染 -->
  52.     <p>
  53.       <router-link :to="{path:'/header'}" @click.native="mouseover">go header(event)</router-link>
  54.     </p>
  55.  
  56.     <!-- 路由出口 -->
  57.     <!-- 路由匹配到的组件将渲染在这里 -->
  58.     <router-view></router-view>
  59.   </div>
  60.   <script>
  61.     //0. 如果使用模块化机制编程，导入Vue和VueRouter，要调用Vue.use(VueRouter)
  62.  
  63.     //1. 定义路由组件
  64.     //可以从其他文件 import 进来
  65.     const Footer = { template: '<div>footer</div>' };
  66.     const Header = { template: '<div>Header</div>' };
  67.  
  68.     //2. 定义路由
  69.     //每个路由应该映射一个组件。其中”compoment“ 可以是：
  70.     //通过Vue.extend() 创建的组件构造器
  71.     //或者，只是一个组件对象
  72.     const routes = [
  73.       { path: '/header', component: Header },
  74.       { path: '/footer', component: Footer }
  75.     ]
  76.  
  77.     //3. 创建router实例，然后传 `routes` 配置
  78.     const router = new VueRouter({
  79.       routes
  80.     })
  81.     var app = new Vue({
  82.       el: '#root',
  83.       router,
  84.       methods: {
  85.         mouseover: function () {
  86.           console.log(1111);
  87.         }
  88.       },
  89.     })
  90.   </script>
  91. </body>
  92.  
  93. </html>
